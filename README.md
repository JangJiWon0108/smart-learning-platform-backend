# 정보처리기사 지능형 학습 플랫폼

정보처리기사 실기 시험 대비를 위한 AI 기반 학습 플랫폼
문제 해설, 유사 문제 추천, 코드 실행 흐름 시각화 서비스 제공

---

## 핵심 기능

### 1. 지능형 문제 풀이 (Solver Agent)

텍스트 질문 또는 문제 사진(OCR 업로드)을 입력하면 즉시 해설을 제공합니다.

- **멀티모달**: 순서도, 코드 스크린샷 등 이미지를 분석하여 텍스트로 변환 후 해설
- **과목 태깅**: 실기 과목(예: 프로그래밍 언어 활용) 자동 분류 및 관련 이론 연결
- **최신 출제 경향 반영**: Google Search 기반 실시간 웹 검색으로 신규 IT 용어 및 출제 패턴 대응

---

### 2. 맞춤형 유사 문제 추천 (RAG Curator)

사용자의 취약 유형을 분석하여 관련 기출 문항을 추출·추천합니다.

- **벡터 검색**: 키워드(예: `C언어 2중 포인터`, `자바 업캐스팅`) 및 난이도 기반 정밀 필터링
- **로직 구조 유사도 매칭**: 단순 키워드 매칭이 아닌 문제의 논리 구조 기반 우선순위 정렬

> 데이터 소스: 정보처리기사 실기 기출 문항 (2020~2025, 2026-1)

---

### 3. 멀티 언어 실행 흐름 시각화 (Language Tracer)

C, Java, Python 코드의 실행 과정을 메모리 수준에서 단계별로 시각화합니다.

| 언어 | 시각화 요소 |
|------|-------------|
| **C** | 포인터가 가리키는 메모리 주소(Address)와 값(Value)을 칸 형태로 표시. 2중 포인터는 화살표 연결선으로 표현 |
| **Java** | `new` 키워드 시 Heap 영역 인스턴스 생성 과정, `super()` 호출에 따른 부모-자식 이동 경로 애니메이션 |
| **Python** | 변수가 가리키는 객체 ID 변화 추적, 리스트 슬라이싱 범위 직관적 표시 |

**UI 컴포넌트**
- **Step Player**: 라인별 재생 / 일시정지 / 이전 단계 제어
- **Variable Monitor**: 현재 실행 라인에서 값이 변하는 변수 하이라이트

---

## 시스템 아키텍처

**Google ADK 2.0**의 그래프 기반 라우팅 패턴을 사용합니다. 의도 분류 에이전트가 질문을 **해설** · **유사 문제 추천** · **실행 흐름 시각화** · **기타**로 나누고, 라우터가 해당 경로로 분기합니다. 유사 문제 추천 경로와 실행 흐름 경로는 내부에서 병렬로 퍼진 뒤 **병합 노드**에서 다시 합쳐집니다.

### 파이프라인 개요

| 단계 | 구성 요소 | 역할 |
|------|-----------|------|
| 1 | 질문 저장 | 원본 질문을 상태에 보관 |
| 2 | 의도 분류 에이전트 | 의도 출력 스키마로 분류 |
| 3 | 의도 라우터 | 분류 결과에 따라 하위 처리로 연결 |
| 4 | 경로별 처리 | 아래 텍스트 트리 참고 |
| 5 | 결과 렌더링 | 해설 / 추천 문제 / 시각화 애니메이션 |

### 에이전트 · 도구 매핑

| 경로 | 에이전트·함수 | 도구 |
|------|---------------|------|
| 해설 | 해설 에이전트 | 실시간 웹 검색 |
| 유사 문제 추천 | 기출 큐레이터 에이전트 | 벡터 검색 |
| 실행 흐름 시각화 | 실행 흐름 시각화 에이전트 | 코드 샌드박스 |
| 기타 | 폴백 안내 | 없음 |

### 텍스트 트리

```
[사용자 입력: 텍스트 / 이미지]
        │
        ▼
질문 저장 ── 원본을 상태에 저장
        │
        ▼
의도 분류 에이전트 ── 해설 | 유사 문제 추천 | 실행 흐름 시각화 | 기타
        │
        ▼
    의도 라우터
        │
        ├── 해설 ──────────► 해설 전처리(이미지·과목) ──► 해설 에이전트 (도구: 실시간 웹 검색)
        │
        ├── 유사 문제 추천 ──► 질문 임베딩 ──► 취약·오답 분석
        │                         │
        │                         ▼
        │                   유사 문제 병합 노드 ──► 검색 결과 병합·난이도 필터
        │                         │
        │                         ▼
        │                   기출 큐레이터 에이전트 (도구: 벡터 검색)
        │
        ├── 실행 흐름 시각화 ──► 언어 감지(C·자바·파이썬) ──┐
        │                        코드 파싱·실행 라인 추출 ─┼──► 실행 흐름 병합 노드
        │                                                    ▼
        │                                              언어별 실행 단계 병합
        │                                                    ▼
        │                                              실행 흐름 시각화 에이전트 (도구: 코드 샌드박스)
        │
        └── 기타 ────────────► 안내 메시지 폴백

        해설 / 유사 문제 추천 / 실행 흐름 시각화 / 기타 ──► [결과 렌더링]
```
